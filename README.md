## 🗂️ Learning Roadmap

### 🔹 Phase 1 — SQL Fundamentals
- SELECT, WHERE, ORDER BY
- DISTINCT, LIMIT, OFFSET
- Constraints (PK, FK, UNIQUE)
- NULL handling

---

### 🔹 Phase 2 — Aggregations
- COUNT, SUM, AVG, MIN, MAX
- GROUP BY
- HAVING
- Real interview questions

---

### 🔹 Phase 3 — SQL JOINs (🔥 Most Important)
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN
- SELF JOIN

---

### 🔹 Phase 4 — Subqueries
- Scalar subqueries
- Correlated subqueries
- EXISTS vs IN
- NOT EXISTS

---

### 🔹 Phase 5 — Window Functions (Advanced SQL)
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- LEAD() / LAG()

---

### 🔹 Phase 6 — Database Design
- Normalization (1NF, 2NF, 3NF)
- Denormalization
- Entity relationships
- Real system design schemas

---

### 🔹 Phase 7 — Indexing & Performance
- Index basics
- Composite indexes
- Query optimization
- EXPLAIN plan

---

### 🔹 Phase 8 — Transactions & Concurrency
- ACID properties
- Isolation levels
- Locks & deadlocks

---

### 🔹 Phase 9 — Real Interview Problems & leetcode 50 Questions
- Second-highest salary
- Duplicate records
- Top N per group
- Pagination queries
